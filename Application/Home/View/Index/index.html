【公司简介】
企业简介Company Profile

贵州大黔朝酒业有限公司是位于“中国第一酒镇”茅台镇，位于贵州省遵义仁怀市赤水河畔，是川黔水路交通的咽喉要地，古有“川盐走贵州，秦商聚茅台”的写照，茅台镇是中国酱酒圣地，域内白酒业兴盛。1995年8月，大黔朝烧坊正式成立；1999年6月，大黔朝烧坊正式更名大黔朝酒厂，投入巨资在茅台镇上坪村倾力打造酱香型白酒生产基地，基地占地面积约300亩，年基酒生产能力4500吨，基酒储存能力15000吨，所有基酒产品完全按照茅台镇传统的生产工艺要求，严格按照节气：端午小麦采曲、重阳高粱投料，全年分两次投料，同一批料要经九次蒸煮、加曲、堆积发酵，一年一个生产周期，在此期间内，经高温制曲、高温堆积、高温入池、高温接酒、低糖化制曲、低水分入池、低出酒量、低酒精浓度等工艺...
